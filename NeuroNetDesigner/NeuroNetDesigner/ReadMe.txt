Папка Creatures - содержит файлы с описанием моделей
Папка Nets - содержит файлы с описанием нейронных сетей
В папке Res создаются отдельные папки для каждой нейронной сети при ее запуске.
В этих папках хранятся результаты запуска:
*_cur_creature.txt - текущие координаты и состояния модели
*_dist.txt - (обучение) для каждого тика сохраняется расстояние, на которое он сдвинулся от начальной координаты
*_run_dist.txt - (запуск без обучения) для каждого тика сохраняется расстояние, на которое он сдвинулся от начальной координаты
*_weights_and_biases.txt - ПОСЛЕДНИЕ веса и смещения
Все эти данные, кроме пройденного расстояния, записываются в файл каждые 100 тиков

Запускать нужно файл NeuroNetDesigner.exe
